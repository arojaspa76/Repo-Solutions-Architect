# Introducción al Diseño de Infraestructura para LLMs

## Objetivo

Conocer los conceptos teóricos y prácticos del diseño de infraestructura para modelos de lenguaje de gran escala (LLMs), sentando las bases para los capítulos siguientes.

---

## Contenido

### Conceptos Básicos

- Introducción al curso y contexto general.
- Requisitos de infraestructura para LLMs (cómputo, memoria, red, almacenamiento).
- Plataformas de nube: AWS, Azure y GCP — comparativa general.

### Casos de Uso Empresariales

- Aplicaciones de LLMs en la industria (atención al cliente, generación de contenido, análisis de documentos, etc.).
- Análisis de requisitos a partir de un caso práctico real.

---

## Requisitos Previos

- Conocimientos básicos de computación en la nube.
- Cuenta activa en al menos una plataforma (AWS, Azure o GCP).
- Python 3.10+ instalado localmente (para los notebooks).
